## Prise en main

### lancer l'app 

* sur windows s'assurer que le docker desktop a bien lancé son daemon (démarrer docker desktop)

```bash
docker compose up -d
```

### arreter les services

```bash
docker compose down
```

## Les Services

### Backend

adresse : [backend](http://localhost:1337)

On utilise npm comme gestionnaire de packets

### Frontend

adresse : [frontend](http://localhost:3000/)
On utilise yarn comme gestionnaire de packets
